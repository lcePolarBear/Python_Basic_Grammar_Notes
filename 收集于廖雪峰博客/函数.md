## 函数的知识点

* 定义
```
def my_abs(x):
    if x >= 0:
        return x
    else:
        return -x
```

* 如何调用外部文件的函数
```
from filename import functionname
functionname()
```

* 占位符 pass
```
def nop():
    pass
```
```
if age >= 18:
    pass
```

* 实现参数检查
```
def my_abs(x):
    if not isinstance(x, (int, float)):
        raise TypeError('bad operand type')
    if x >= 0:
        return x
    else:
        return -x
```
