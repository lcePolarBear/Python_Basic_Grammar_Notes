# 模板
## 模板继承
- 定义一个基础模板 `templates/base.html` ，这个页面存放整个网站共用的内容，并预留 block 子模版用于差异化内容
    ```html
    <!-- base.html -->
    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>{% block title %}    {% endblock %}
        </title>
    </head>
    <body>
    <nav>
        顶格
    </nav>
    <div>
    {% block www %}

    {% endblock %}
    </div>
    <footer>
        底格
    </footer>
    </body>
    </html>
    ```
- 在子模版继承这个模板，并填充预留区域内容
    ```html
    <!-- index.html -->
    {% extends 'base.html' %}
    {% block title %}
    block title
    {% endblock %}
    {% block www %}
    www
    {% endblock %}
    ```
